VAMOS VER COMO FUNCIONA O BOX DE UM H1

também de um span, que é inline, assim como um link qualquer

uma imagem não é box level, pois tem o display inline-block mas um parágrafo é - uma citação quote não é box level

mas um blockquote é box level
quando temos um elemento inline level e queremos centralizá-lo, basta text-align="center"

para centralizar block levels como esse parágrafo precisamos do margin: auto;

Aqui vai o short-hand notation para border padding margin e outline 

        border: 10px solid black;
        padding: 10px 10px 10px 14px; 
                                    /*se for só um valor vale par todas as direções --> top right bottom left
                                    se for dois valores o 1º vai para top e bottom e o segundo para left e right*/
        margin:20px 25px 20px 25px;
                                     /*para centralizar com shorthand só algumas direções, podemos usar auto como valor, assim */
        margin: 20px auto 40px auto;
        outline: 5px dashed salmon;

        o shorthand é sempre: SIZE STYLE COR
