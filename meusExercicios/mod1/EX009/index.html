Estilos em Cascata - As CSS

Formas de incorporação

Inline

As CSS inline são insideridas quando adicionamos o atributo style="atributo:valor,valor; atributo:valor;" em uma tag. Elas devem ser evitadas. O idela é usá-la somente em alterações muito pontuais, como acabamos de fazer aqui.Esse estilo pontual foi feito usando o seguinte código - style="text-decoration: underline; color: red; font-size: larger;"

Interno

As CSS internas são adicionadas pela tag style aberta no head do documento html. Elas são boas por separar as declarações de estilo do código HTML e permitir o uso de seletores, ids, classes, etc.. No entanto, se for necessário aplicar estilos a mais de uma página, utilizar esse método pode ser demasiado trabalhoso e suscetível a erros. Abaixo temos o código CSS que utilizamos para configurar essa página

                    
                        <head>
                        <style>
                            body{
                                background-color: #cef5df;
                                font-size:16pt;
                                }
                            p{ text-align: justify;
                                font-size: 0.8em       <!--unidade 'em' para valores no css = unidade reltiva propocional ao valor herdado, nesse caso,
                                                       80% do valor 16pt, configurado no seletor body-->
                            }
                            header{
                                text-align:center;
                                margin-bottom: 200px;
                            }
                            img{
                                position:absolute;
                                right:15px;
                                top:15px;
                                width: 150px;
                                height:150px;
                            }
                            li {
                                border: solid 1px grey;
                                padding: 10px;
                            }
                            ul, ol{
                                width: 35%;
                                height:35%;
                            }
                        </style>
                        </head>

Externo

Para adicionar CSS externo, devemos adicionar uma tag link no head do html, dessa forma <link rel="stylesheet" href="nome_do_arquivo.css">. Assim, todo o código referente a estilo ficará completamente separado em outro arquivo. Isso facilita a escrita de todo o design do site, pois as linhas de css frequentemente são muitas e tomam muito espaço se forem deixadas junto do arquivo HTML. Além disso, o externo facilita a aplicação de um mesmo estilo a varios arquivos.

É também possível linkar vários arquivos CSS no mesmo HTML, assim eles se complementam. Neste documento as bordas dos elementos h3 foram feitas em um arquivo css, enquanto a formatação dos códigos-fonte foi feita com outro arquivo css.

Código Externo 1 - <link rel="stylesheet" href="style1.css">

                            @charset "utf-8";
                            code{
                                color:rgb(79, 69, 69);
                                font-style: italic;
                                font-size: 12pt;
                            }

Código Externo 2 - <link rel="stylesheet" href="style2.css">

                            @charset "utf-8";
                            h3{
                                border: ridge 1px blue;
                                width:200px;
                            }

Ordem de Prioridade das CSS

É possível utilizar todos os modos supracitados para inserir CSS no projeto. No entanto, eles têm uma ordem de prioridade, caso existam diferentes declarações sobre os mesmos seletores ou elementos. A ordem é a seguinte:

  1. CSS INLINE
  2. CSS INTERNO
  3. CSS EXTERNO
